# Data Model: 项目检查与Docker部署验收

本项目为部署验收项目，数据模型已在主项目 `kitty-specs/001-xianyu-auto-management-phase1/data-model.md` 中定义。

## 引用数据模型

本项目直接使用主项目的数据模型，无需新建数据表。

### 核心数据表

| 表名 | 说明 |
|------|------|
| accounts | 咸鱼账号 |
| products | 商品 |
| card_keys | 卡密 |
| orders | 订单 |
| messages | 消息 |
| alerts | 告警 |
| settings | 系统配置 |
| logs | 运行日志 |
